## 令和2年度 国勢調査データの可視化と地域分析

令和2年度の国勢調査（e-Stat）のデータを用いて、日本の人口分布や都道府県別の男女比を分析・可視化しました。

## 概要
1万行を超える統計データをPythonを用いて整理し、人口ランキングや地域ごとの人口構成の違いを分析しました。
特に地元の愛知県と東京都を比較し、産業構造が人口比率に与える影響について考察しています。


## 分析内容
* **データの前処理**：カンマを含む文字列データの数値化、欠損値の処理
* **人口分布の可視化**：都道府県別の人口ランキング、上位都府県のシェアを円グラフで表示
* **男女比の算出**：地域ごとの男女比（性比）の計算と、全国平均との比較

## 使用技術
* **言語**：Python 3.13.7
* **ライブラリ**：pandas, matplotlib
* **データソース**：[e-Stat（政府統計の総合窓口）](https://www.e-stat.go.jp/)

## 意識したこと・学んだこと
* **論理的な分析プロセスの構築**：行き当たりばったりにコードを書くのではなく「前処理 → 全体俯瞰（ランキング、円グラフ） → 特定地域の深掘り（男女比）」と順序立てて分析を進めることを意識しました。これにより、大きなデータから段階的に意味のあり情報を抽出する流れを学ぶことができました。
* **データクリーニングの正確性**：統計データ特有のカンマ入り文字列を数値型に変換する際、エラーが多発しました。デバッグを通じて、変素の状態を正しく把握することの重要性を実感しました。
* **仮説の検証**：「製造業が盛んな地域は人口構成に特徴がでるのではないか」という仮説を立てて分析した結果、愛知県の数値（99.5）にその特性が現れていることが確認できました。
